# kild
Mini framework para o desenvolvimento de aplicações simples

Usar um framework é muito útil nos dias atuais. Onde o tempo conta muito para o desenvolvimento de qualquer aplicação.

Sabendo que existem aplicações que não necessitam de um Framework potente como Laravel ou Zend.
Pensando nisso, foi desenvolvido esse mini-framework para auxiliar no desenvolvimento de aplicações web com qualidade e rapidez sem desperdício de recursos.

Versão : 0.1 - 2017
Autor : Edigleysson Silva <edigleyssonsilva@gmail.com>

Core :

- Roteamento 
- Models com ORM
- Interface amigável para manipulação de banco de dados
